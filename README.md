# 📖 Aethelgard: The Architect's Grimoire

> **"A Warrior only compares himself against yesterday's version."**

Aethelgard is a high-fantasy, local-first RPG productivity tool designed to transform the "Effort" of daily life into the "Value" of a digital legacy. Built for developers, students, and seekers who want to gamify their growth.

## 📜 The Vision
Aethelgard moves away from clinical, flat "to-do" lists and replaces them with a **Physical Grimoire**. Every task is a contract; every completion is a wax-sealed victory.

- **Dopamine-Driven Progress:** Variable ratio rewards (Loot Chests) and visual "Slam" animations.
- **Stat-Based Growth:** Tasks map to four core attributes: Vitality, Wisdom, Fortune, and Charisma.
- **The Chronicler:** (In-Dev) AI-powered narrative that turns your week into a high-fantasy saga.

---

## 🛠️ Tech Stack
This project is built with modern web technologies focused on performance and immersion:

* **Framework:** [React](https://react.dev/) + [Vite](https://vitejs.dev/)
* **Language:** [TypeScript](https://www.typescriptlang.org/) (Strict Mode)
* **State Management:** [Zustand](https://github.com/pmndrs/zustand) (Persistent Local Storage)
* **Animations:** [Framer Motion](https://www.framer.com/motion/)
* **Deployment:** [Cloudflare Pages](https://pages.cloudflare.com/)
* **Styling:** Custom CSS with Texture Mapping & High-Fantasy Typography

## ⚖️ The Mechanics

### The Leveling Formula
Aethelgard uses a quadratic progression curve to ensure early levels feel rewarding while late-game levels require true mastery:
$$XP_{Required} = n^2 \times 50 + n \times 50$$
*Where $n$ is the target level.*

### Attribute Mapping
- **Vitality (Red):** Health, Sleep, Exercise.
- **Wisdom (Blue):** Coding, French Study, Reading.
- **Fortune (Gold):** SaaS Development (lleno.io), Career, Finance.
- **Charisma (Purple):** Music (Jazz/Guitar), Social, Networking.
